# 手动添加

手动添加 Apron React Bits 组件，按照你的习惯来管理组件，没有入侵性。

## 步骤

### 挑选你想用的组件

预览所有组件，选中你需要用到的。

### 安装依赖

部分组件需要依赖 `gsap`，会在组件文档页面上注明。如果需要，请运行：
```bash
npm install gsap
```

请放心，组件页面上也会有这行指令。

### 复制代码

依次把页面上 `tsx` 和 `scss` 的代码复制到你的项目中。如果你使用 `css`，那么可以使用 AI 工具帮你转一次。

### 使用组件

按照正常的组件引用即可。每个组件都会在页面上有一个完成的配置示例代码、属性表格，跟随文档配置即可。

## 就这么多

Apron React Bits 使用这样的方式尽可能地减少对你的代码的入侵性，同时你也可以随意修改使用。
