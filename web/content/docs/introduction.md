# 介绍

Apron React Bits 是一个开源的 UI 组件集合，精心设计，旨在增强您的 React Web 应用程序。

这不是典型的组件库，这意味着你在这里找不到一组通用的按钮、输入框或其他常见的用户界面元素。

基本上，这些组件旨在通过为您的项目增添创意，帮助您脱颖而出，并在视觉上引人注目。

## 使命
Apron React Bits 的目标很简单：提供灵活、视觉效果惊艳且最重要的是免费的组件，将 Web 项目提升到一个新的水平。

为了实现这一目标，该项目致力于遵循以下原则：

- **免费使用**：代码归您所有，您可以免费将其用于您的项目中。
- **属性优先策略**：通过精心设计的属性实现轻松定制
- **完全模块化**：仅安装所需组件，Apron React Bits 不是依赖项。

### 自由混战
您选择引入到项目中的每个组件都可以进行修改或扩展，因为您可以完全查看代码，而不仅仅是导入。

### 属性优先方法
每个组件都设计得灵活且可定制，通过属性可以调整外观和感觉，而无需总是深入代码。

### 全模块化
Apron React Bits 不是经典的 NPM 库，你只需通过复制代码或使用 CLI 安装所需的组件，而无需引入整个库。



## 表现
虽然我们尽一切可能优化组件并提供最佳体验，但在使用 React Bits 时，请记住以下几点：

- **少即是多**：页面上使用超过 2-3 个组件是不建议的，这可能会导致页面动画过多，从而影响性能或用户体验。
- **移动端优化**：考虑禁用移动端的某些特效，并用静态占位符代替。
- **充分测试**：您的设备或许高端，但也要顾及用户：正式上线前务必在多台设备上进行测试。

## 支持的浏览器

Apron React Bits 支持最近两个版本的浏览器。

如果您需要支持旧版本的浏览器，请自行添加 Babel 和相应的 Polyfill。

由于 React16 不再支持 IE11，因此 Apron React Bits 也不支持 IE 浏览器。
