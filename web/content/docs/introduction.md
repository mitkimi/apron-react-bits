# 介绍

Apron React Bits 是一个轻量级的 React 组件库，提供了一系列高质量、可复用的 UI 组件，帮助开发者快速构建现代化的 Web 应用。

## 特性

- **轻量级**: 按需加载，最小化包体积
- **易用性**: 简单直观的 API 设计
- **可定制**: 支持主题定制和样式覆盖
- **无障碍**: 遵循 WAI-ARIA 标准，支持键盘导航
- **响应式**: 适配各种屏幕尺寸

## 技术栈

- React 18+
- TypeScript
- SCSS
- Next.js (可选)

## 支持的浏览器

- Chrome 64+
- Firefox 67+
- Safari 12+
- Edge 79+
